# Farmacia Jiménez Rosales - Landing Page

![Farmacia Jiménez Rosales](src/assets/logo.png)

Una landing page moderna y elegante diseñada para la **Farmacia Jiménez Rosales**, enfocada en proporcionar una experiencia de usuario fluida y acceso rápido a información crítica como ubicación, contacto y servicios.

> **Lema:** "Tu salud en buenas manos"

---

## 🚀 Tecnologías

Este proyecto está construido con un stack moderno enfocado en el rendimiento y la mantenibilidad:

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Estilos:** Vanilla CSS con [CSS Modules](https://github.com/css-modules/css-modules)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
- **Iconos:** [Lucide React](https://lucide.dev/)
- **Testing:** [Vitest](https://vitest.dev/) & [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)

---

## ✨ Características Principales

- **Estado del Negocio en Tiempo Real:** Indicador dinámico de "Abierto" o "Cerrado" basado en el horario actual.
- **Acceso Rápido a Ubicación:** Integración directa con Google Maps para navegación GPS.
- **Conversión Directa vía WhatsApp:** Botón de contacto rápido para consultas y pedidos.
- **Catálogo de Servicios:** Sección dedicada a Medicamentos, Vitaminas y Delivery.
- **Diseño Elegante y Responsivo:** Interfaz optimizada para todos los dispositivos con una paleta de colores profesional (Azul, Celeste y Blanco).
- **Animaciones Fluidas:** Transiciones suaves que mejoran la percepción de calidad del sitio.

## 🛠️ Instalación y Desarrollo

### Prerrequisitos
- Node.js (v18 o superior recomendado)
- npm o pnpm

### Pasos
1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/jcano37/pharmacy-jr.git
   cd pharmacy-jr
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Ejecutar pruebas:**
   ```bash
   npm run test
   ```

5. **Construir para producción:**
   ```bash
   npm run build
   ```
